快速开始

本指南将帮助你快速上手 GM-Bench，开始评测大模型的国密能力。

环境要求

  • Python ≥ 3.11
  • pip 包管理工具
  • 可选：gm-agent-stack Mock 服务（用于 C5/C6 评测）
注意：开发机 Python 版本低于 3.11 时，harness 代码需要安装 Python 3.11+。C5/C6 需要额外运行 gm-agent-stack Mock 服务。

1. 安装依赖

git clone https://github.com/kintaiW/gm-llm-bench
cd gm-llm-bench
pip install -r requirements.txt

requirements.txt 内容：

anthropic>=0.40.0
openai>=1.50.0
httpx>=0.27.0

2. 仅评测 C1-C4（无需 Mock，最简单）

GPT-4o

export OPENAI_API_KEY=sk-...
python harness/run_eval.py \
  --model gpt-4o \
  --categories C1,C2,C3,C4 \
  --output leaderboard/results/gpt-4o.json

Claude Opus

export ANTHROPIC_API_KEY=sk-ant-...
python harness/run_eval.py \
  --model claude-opus-4-7 \
  --categories C1,C2,C3,C4

DeepSeek

export DEEPSEEK_API_KEY=sk-...
python harness/run_eval.py \
  --model deepseek-chat \
  --categories C1,C2,C3,C4

3. 评测 C5/C6（需要 Mock 服务）

第一步：启动 SVS Mock（来自 gm-agent-stack）

cd ../mp-mock/0029-svs-mock
cargo run -- --mode both &
# Mock 默认监听 http://localhost:9000

第二步：运行完整评测

export SVS_MOCK_URL=http://localhost:9000/mcp
python harness/run_eval.py \
  --model gpt-4o \
  --categories C1,C2,C3,C4,C5,C6 \
  --svs-mock-url $SVS_MOCK_URL

export OPENAI_API_KEY=sk-...

调试模式（只跑前 5 题）

python harness/run_eval.py \
  --model gpt-4o \
  --categories C1 \
  --max-questions 5 \
  --sleep 0

常见问题

Q: 评测速度如何？

A: C1-C4 约 190 题，每个模型需要 10-15 分钟；C5-C6 约 110 题，需要 20-30 分钟。

Q: 如何添加新模型？

A: 见 评测接口 页面的添加新模型部分。

Q: Mock 服务必须启动吗？

A: 只评测 C1-C4 时不需要；评测 C5-C6 时需要启动 SVS Mock。
