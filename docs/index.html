什么是 GM-Bench？

GM-Bench 是全球首个面向中国商用密码合规（密评）的大模型评测集，用 Go 编写。它衡量 AI 在国密领域从"知道"到"能做"的完整能力链。

核心价值：GM-Bench 不只是选择题集，它通过可执行验证来衡量 LLM 的国密能力，包括代码改写和工具调用的实际运行验证。

评测维度

C1 - 条款归属

60 题，单选题，考察 AI 能否将密评要求归属到正确标准条款。

C2 - 标准映射

40 题，匹配题，考察 AI 能否把设备/协议/算法对应到正确 GM/T 编号。

C3 - SKOD 评分

50 题，数值 JSON，考察 AI 能否对实际系统做出合理的四维密评打分。

C4 - 不符合项分级

40 题，多选 + 严重程度，考察 AI 能否识别不符合项并判断严重/一般/轻微。

C5 - 代码改写

30 题，代码生成，考察 AI 能否生成可跑通的国密迁移代码。

C6 - 工具调用

80 题，MCP 调用序列，考察 AI 能否选择正确的 MCP 工具序列完成任务。

关键特性

  • 可执行验证：C5 和 C6 用代码说话，实际运行对接 SVS/SDF/SKF Mock 设备
  • 标准覆盖：基于 GB/T 39786-2021 和多个 GM/T 标准
  • 多模型支持：支持 GPT-4o、Claude Opus、DeepSeek 等主流模型
  • 排行榜系统：自动生成 HTML 排行榜页面
  • 易于扩展：支持添加新模型和新题目

快速开始

# 安装依赖
git clone https://github.com/kintaiW/gm-llm-bench
cd gm-llm-bench
pip install -r requirements.txt

# 评测 C1-C4（无需 Mock）
export OPENAI_API_KEY=sk-...
python harness/run_eval.py \
  --model gpt-4o \
  --categories C1,C2,C3,C4 \
  --output leaderboard/results/gpt-4o.json

# 查看结果
python scripts/update_leaderboard.py
# 生成 leaderboard/index.html

项目结构

gm-llm-bench/
├── data/
│   └── questions/         # 评测题目
├── harness/
│   ├── run_eval.py        # 主入口
│   ├── evaluators/        # 评分器
│   └── providers/         # 模型提供器
├── leaderboard/
│   ├── index.html         # 排行榜页面
│   └── results/           # 评测结果
├── scripts/
│   ├── run_all_models.sh  # 批量评测脚本
│   └── update_leaderboard.py # 生成排行榜
└── requirements.txt

配套项目

  • 0029-svs-mock - 签名验签服务器模拟器（C5/C6 验证用）
  • 0018-sdf-mock - 服务器密码机模拟器（C5/C6 验证用）
  • 0016-skf-mock - 智能密码钥匙模拟器（C5/C6 验证用）
  • ccasa-harness - AI 密评助手（使用 GM-Bench 评测）
